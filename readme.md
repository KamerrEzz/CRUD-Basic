# CRUD Basicos para principantes.

Este es una pequeña CRUD api, creado en un taller de mi comunidad.

## Inicia
```js
npm i
npm run start
```

## rutas
```js
GET » /api/usuarios
GET » /api/usuarios/1234567
POST » /api/usuarios
PUT » /api/usuarios/1234567
DELETE » /api/usuarios/1234567
```

## Estructura
```js
[+] models
    [-] usuarios
[+] routes
    [-] usuarios
[+] services
    [-] usuarios
```
